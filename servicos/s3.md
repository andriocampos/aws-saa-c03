# S3 — Simple Storage Service

## Storage Classes

| Classe | Disponibilidade | Retrieval | Uso |
|--------|----------------|-----------|-----|
| Standard | 99,99% (multi-AZ) | Imediato | Acesso frequente |
| Standard-IA | 99,9% (multi-AZ) | Imediato | Acesso infrequente, multi-AZ |
| One Zone-IA | 99,5% (single-AZ) | Imediato | Acesso infrequente, pode recriar |
| Intelligent-Tiering | 99,9% | Imediato | Padrão de acesso desconhecido |
| Glacier Instant Retrieval | 99,9% | Milissegundos | Arquivos, acesso trimestral |
| Glacier Flexible Retrieval | 99,99% | Minutos a horas | Backups de longo prazo |
| Glacier Deep Archive | 99,99% | Até 12 horas | Retenção de 7-10 anos |

## Segurança

- **Bucket Policy:** resource-based, controla acesso ao bucket e objetos
- **ACLs:** legacy, preferir Bucket Policies
- **Block Public Access:** configuração que sobrepõe policies e ACLs
- **Pre-signed URLs:** acesso temporário a objetos (upload ou download)
- **OAC (Origin Access Control):** permite CloudFront acessar S3 privado

## Replicação

| Tipo | Descrição |
|------|-----------|
| CRR (Cross-Region Replication) | Replica entre regiões — compliance, latência |
| SRR (Same-Region Replication) | Replica na mesma região — log aggregation, live replica |

- Requer versionamento habilitado em ambos os buckets
- Objetos existentes antes da configuração NÃO são replicados automaticamente

## S3 Object Lock (WORM)

| Modo | Descrição |
|------|-----------|
| Governance Mode | Admin pode sobrescrever/deletar com permissão |
| Compliance Mode | Ninguém pode sobrescrever/deletar, nem root |

## Multipart Upload

- Recomendado para arquivos > 100MB
- Obrigatório para arquivos > 5GB
- Melhora throughput e resiliência de uploads

## Diferenças Críticas

- **Standard-IA vs One Zone-IA:** One Zone-IA é mais barato mas dados perdidos se AZ falhar
- **Glacier Instant vs Flexible Retrieval:** Instant é mais caro mas acesso imediato; Flexible é mais barato com latência de minutos a horas
- **S3 Transfer Acceleration:** usa CloudFront Edge Locations para acelerar uploads de longas distâncias
