## Dica da semana
CloudWatch vs CloudTrail vs Config: cada um responde uma pergunta diferente.
"Está funcionando?" → CloudWatch | "Quem fez?" → CloudTrail | "Está correto?" → Config.
ECS Fargate vs Lambda: Fargate para containers de longa duração; Lambda para funções curtas.
